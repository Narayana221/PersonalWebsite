# Professional Portfolio Website

A modern, developer-first portfolio website built with Angular 17+ and Tailwind CSS, featuring a clean, high-contrast design optimized for performance.

## 🎨 Design Philosophy

- **Developer-First**: Clean, readable code with semantic HTML
- **High-Contrast**: Midnight Blue, Slate, and Emerald Green color palette
- **Performance-Oriented**: Optimized bundle sizes and lazy loading
- **Responsive**: Mobile-first design that scales beautifully

## 🚀 Getting Started

### Installation

```bash
cd portfolio-app
npm install
npm start
```

Visit `http://localhost:4200/`

### Build for Production

```bash
npm run build
```

## 📝 Customization

Edit `app.component.ts` to update your personal information, projects, skills, and experience.
